# GeoCosmos：3D 地球空间与环境态势看板

这是一个可直接运行的前端程序：

- 打开后显示宇宙背景下的 3D 地球（Three.js）。
- 接入多个公开 API，将地球空间与环境数据叠加到球体上：
  - **USGS**：全球地震（最近 24 小时）。
  - **NASA EONET**：全球自然事件（最近 30 天）。
  - **Open-Meteo**：多个城市实时天气与降水（用“雨粒子”效果可视化）。
  - **WhereTheISS.at**：国际空间站（ISS）实时位置。

## 运行方式

> 需要联网，因为纹理和数据来自公开 API。

1. 在项目目录下启动静态服务：

```bash
python3 -m http.server 8080
```

2. 浏览器打开：

```text
http://localhost:8080
```

## 可视化说明

- **地震**：按震级显示不同颜色与高度的锥形标记。
- **自然事件**：用蓝色脉冲点标记事件位置。
- **天气**：监测多个典型城市，降雨/降水时叠加雨粒子效果。
- **ISS**：黄色锥形标记当前地表投影点。

## 交互

- 左键拖拽：旋转
- 右键拖拽：平移
- 滚轮：缩放
- 点击面板按钮“刷新数据”：重新拉取所有 API
